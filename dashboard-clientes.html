Configuración de fuente de datos

El dashboard lee datos desde tu Google Apps Script. Pega la URL de tu Web App aquí. La URL se guarda en tu navegador (localStorage) y persiste entre sesiones.

Instrucciones para el Apps Script
  1. Abre script.google.com y crea un nuevo proyecto
  2. Pega el contenido del archivo apps-script.gs
  3. Clic en Implementar → Nueva implementación
  4. Tipo: Aplicación web
  5. Ejecutar como: Yo (tu cuenta Google)
  6. Quién tiene acceso: Cualquier persona
  7. Clic en Implementar → autoriza los permisos
  8. Copia la URL generada y pégala arriba
